## Getting Started

### Installing

1. Clone the repo:

```bash
git clone git@github.com:vanya992/Blissful-Nirvana.git
```

2. Install the dependencies:

```bash
npm install
```

### Running

To run the app, run the following commands:

```bash
npm run start
```
